#  Project Challenge: Social Proof Section

Welcome to the **Social Proof Section** challenge! This project focuses on complex layouts, staggered positioning, and responsive design using modern CSS techniques.

##  Project Requirements

To successfully complete this challenge, your solution must meet the following criteria:

* **Responsive Layout:** The section must transition from a single-column mobile view to a multi-column desktop view.
* **Semantic HTML:** Use proper tags like `<section>`, `<article>`, `<blockquote>`, and `<ul>`.
* **Design Accuracy:** Match the colors (Very Dark Magenta, Soft Pink) and typography provided in the `style-guide.md`.

##  How to Get the Starter Materials

1.  **Download:** Go to the [Social Proof Section Challenge](https://www.frontendmentor.io/challenges/social-proof-section-6e0qTv_bA) and click **"Download starter"**.
2.  **Setup:** Extract the files and initialize a Git repository.
3.  **Reference:** Use `style-guide.md` for exact hex codes and font sizes.
